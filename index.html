Tutorial CorelDRAW, Photoshop, dan AutoCAD

Panduan Penggunaan CorelDRAW

CorelDRAW adalah perangkat lunak desain grafis yang populer digunakan untuk berbagai kebutuhan desain, mulai dari logo hingga ilustrasi kompleks.

Memulai Proyek Baru

  1. Buka CorelDRAW: Luncurkan aplikasi CorelDRAW dari desktop atau menu Start.
  2. Buat Dokumen Baru:
    • Pilih "New Document".
    • Tentukan ukuran, resolusi, dan mode warna.
    • Klik "OK" untuk memulai.

Mengenal Antarmuka CorelDRAW

  • Menu Bar: Berisi menu seperti File, Edit, View, dll.
  • Toolbox: Berisi alat-alat menggambar dan mengedit.
  • Property Bar: Menampilkan opsi sesuai alat yang dipilih.
  • Color Palette: Mempermudah pemilihan warna.
  • Docker: Untuk pengaturan lanjutan.

Menyimpan dan Mengekspor Proyek

  1. Menyimpan Proyek:
    • Pilih "File" > "Save".
    • Pilih lokasi dan format file (.cdr).
  2. Mengekspor Proyek:
    • Pilih "File" > "Export".
    • Tentukan format seperti PNG, JPEG, atau PDF.
    • Klik "Export".

Panduan Penggunaan Photoshop

Adobe Photoshop adalah perangkat lunak pengeditan grafis yang sangat populer untuk fotografi dan desain digital.

Memulai dengan Photoshop

  1. Unduh dan Instal:
    • Unduh dari situs Adobe.
    • Ikuti petunjuk instalasi.
  2. Membuka Photoshop: Klik ikon Photoshop di desktop atau menu Start.

Menyimpan dan Mengekspor Proyek

  1. Menyimpan Proyek:
    • Pilih "File" > "Save As".
    • Pilih format PSD untuk pengeditan lebih lanjut.
  2. Mengekspor Proyek:
    • Pilih "File" > "Export" > "Export As".
    • Pilih format seperti PNG atau JPEG.

Panduan Penggunaan AutoCAD

AutoCAD adalah salah satu aplikasi perangkat lunak yang paling populer digunakan untuk desain grafis teknik dan arsitektur. Aplikasi ini memungkinkan pengguna untuk membuat gambar 2D dan 3D secara detail dan akurat. Berikut adalah panduan dasar untuk memulai penggunaan AutoCAD.

Pengenalan Antarmuka AutoCAD

  • Menu Baru: Berisi berbagai menu untuk mengakses perintah dan fungsi AutoCAD.
  • Ribbon: Pita yang menampilkan ikon untuk akses cepat ke berbagai alat.
  • Toolbar: Baris alat yang dapat disesuaikan, biasanya berisi perintah yang sering digunakan.
  • Drawing Area: Area utama tempat Anda membuat dan mengedit gambar.
  • Command Line: Tempat Anda dapat mengetik perintah secara langsung.
  • Status Baru: Menampilkan informasi terkini tentang gambar dan opsi tampilan.

Membuat Gambar 2D Dasar

  1. Membuat Garis:
    • Klik ikon "Line" pada ribbon atau ketik 'LINE' di command line, lalu tekan 'Enter'.
    • Klik di drawing area untuk menentukan titik awal garis.
    • Gerakkan kursor dan klik untuk menentukan titik akhir garis.
    • Tekan 'Enter' untuk menyelesaikan perintah.
  2. Menggunakan Perintah Circle:
    • Klik ikon "Circle" atau ketik 'CIRCLE' di command line, lalu tekan 'Enter'.
    • Klik untuk menentukan pusat lingkaran.
    • Gerakkan kursor atau masukkan radius lingkaran, lalu tekan 'Enter'.
  3. Membuat Persegi Panjang:
    • Klik ikon "Rectangle" atau ketik 'RECTANGLE' di command line, lalu tekan 'Enter'.
    • Klik untuk menentukan salah satu sudut persegi panjang.
    • Gerakkan kursor atau masukkan panjang dan lebar, lalu tekan 'Enter'.
